Haldir Gate

Agent identity.
Solved.

Every agent gets a scoped session. Permissions, spend limits, and TTL — enforced on every tool call. No session, no access.

# Create a scoped session for your agent
session = gate.create_session(
  agent_id="research-bot",
  scopes=["read", "browse", "spend:50"],
  ttl=3600
)
Haldir Vault

Secrets agents
never see.

API keys, credentials, and payment limits — encrypted at rest. Agents request access. Vault decides.

# Agent requests a secret — Vault checks scope
key = vault.get_secret(
  "stripe_key",
  session=session
)

# Authorize a payment against the session budget
vault.authorize_payment(session, amount=29.99)
Haldir Watch

Every action.
Recorded.

Immutable audit log for every tool call. Anomaly detection. Cost tracking. The compliance layer enterprises require.

# Every action is logged automatically
watch.log_action(
  session,
  tool="stripe",
  action="charge",
  cost_usd=29.99
)

# Query the audit trail
watch.get_spend(agent_id="research-bot")
