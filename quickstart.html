Step 2 — Install

Step 3 — Run this script

Copy, paste, run. Watch governance happen.

This script creates a session, checks permissions, stores a secret, authorizes a payment, logs to the audit trail, verifies the hash chain, and revokes the session. All live against haldir.xyz.

What just happened

Your agent was governed.

Every action was scoped to a session with permissions and a spend limit. The secret was AES-128 encrypted. The payment was authorized against the session budget. Every action was logged to a hash-chained, tamper-evident audit trail. And when you revoked the session, the agent lost all access instantly.
