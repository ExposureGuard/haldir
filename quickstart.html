Haldir / Quickstart

Get started in 30 seconds

From zero to governed AI agent in 6 steps.

Step 1

Install

pip install haldir
Step 2

Get an API key

curl -X POST https://haldir.xyz/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"name": "my-app"}'

Save the key. You need it for every request.

Step 3

Create a governed session

from sdk.client import HaldirClient

h = HaldirClient(api_key="hld_xxx", base_url="https://haldir.xyz")
session = h.create_session("my-agent", scopes=["read", "spend:50"])

Your agent now has an identity with a $50 budget.

Step 4

Store secrets

h.store_secret("stripe_key", "sk_live_xxx")

AES encrypted. Agents access through scope checks.

Step 5

Use the proxy

h.authorize_payment(session["session_id"], 29.99)
h.log_action(session["session_id"], tool="stripe", action="charge", cost_usd=29.99)

Budget enforced. Action logged. Every time.

Step 6

Check the audit trail

spend = h.get_spend(agent_id="my-agent")
# {"total_usd": 29.99, "by_tool": {"stripe": 29.99}}
